## Markdown Elements

### 1. Headers

Markdown allows you to create headers of various levels:

### 2. Lists

You can use **unordered** lists:

- Lorem
- Ipsum
- Dolor
  - Nested Item 1

Or **ordered** lists:

2. Ipsum
3. Dolor
